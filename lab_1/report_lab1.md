## Задание 5. Раздел и монтирование /disk

### 5.1. Создание нового раздела с помощью fdisk
```bash
sudo fdisk /dev/sda
```
Последовательность действий в fdisk:
1) n (new partition)
2) 4 (partition number)
3) Enter (начальный сектор по умолчанию)
4) Enter (конечный сектор по умолчанию - всё свободное место)
5) w (write changes)

![alt text](img/image-22.png)

### 5.2. Создание нового раздела с помощью fdisk
```bash
sudo lsblk
```

### 5.3. Форматирование нового раздела в ext4
```bash
sudo mkfs.ext4 /dev/sda4
```

![alt text](img/image-23.png)

### 5.4. Создание точки монтирования и монтирование
```bash
sudo mkdir /disk
sudo mount /dev/sda4 /disk
df -h /disk
```

![alt text](img/image-24.png)

### 5.5. Получение UUID нового раздела
```bash
sudo blkid /dev/sda4
```

![alt text](img/image-25.png)

### 5.5. Настройка автоматического монтирования (/etc/fstab)
```bash

```

![alt text](img/image-26.png)

### 5.6. Проверка корректности fstab и выдача прав
